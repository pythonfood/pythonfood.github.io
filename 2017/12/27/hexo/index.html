hexo

hexo常用命令笔记。

安装

npm install hexo -g

升级

npm update hexo -g

初始化

hexo init

显示Hexo版本号

hexo version

新建文章

hexo n "postName"
hexo new "postName"

生成静态文件

hexo g 
hexo generate

启动本地服务

hexo s
hexo server
hexo server -p 5000 #更改端口
hexo server -i 192.168.1.1 #自定义 IP

部署

hexo d 
hexo deploy

草稿

hexo p
hexo publish [layout] <title>

列出网站资料

hexo list <type>

清除db.json和public

hexo clean

新建页面

hexo new page "pageName"
